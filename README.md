# StylishStartup
細字を活用したスタイリッシュ風なkeynoteテンプレートです。ご自由にご活用ください

- Teema_StylishStartup
  - keynoteのテーマファイルです、こちらをお手持ちのkyenoteにインポートしてご利用ください
- StylishStartup.zip
  - 上記テーマをそのままページにしたものです、上記が使えなかった場合にご使用ください（ファイルサイズが重いのでご注意）
  

# 利用環境
keynote 6.6.3　以上
Mac OSX 10.11(El capitan)以上
